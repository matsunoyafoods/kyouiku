📩 LINE配信管理

LINEグループIDの取得方法：
1. LINE DevelopersでBotのWebhook URLを設定: https://あなたのドメイン/api/line/webhook
2. Botを各店舗のLINEグループに招待
3. サーバーログに LINE グループ参加: groupId = Cxxx と表示される
4. そのIDを店舗管理画面に登録
